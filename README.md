# Support Vector Machine Classification Model
Libiries:
```
1. Scikit-learn
2. Seaborn
3. Pandas
4. Matplotlib
```
Key concepts:
1. Target variables definition
2. Data visualization
3. Data splitting into trainnig and test
4. Pairplotting
5. Model trainning
6. Prediction 
7. Accuracy, precision, f1-score and recall metrics

This project is a plant species classification using Support Vector Machine algorithm
based on sepal and petal lenght and width.
